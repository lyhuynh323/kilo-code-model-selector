# Kilo Code Model Selector

Extension VS Code để chọn model cho Kilo Code thông qua Ctrl+Shift+P.

## Cài đặt

1. Mở VS Code
2. Nhấn `Ctrl+Shift+P` và gõ "Developer: Open Folder in Explorer"
3. Mở thư mục `kilo-code-model-selector`
4. Nhấn `F5` để chạy extension trong cửa sổ Extension Development Host

## Cách sử dụng

1. Nhấn `Ctrl+Shift+P`
2. Gõ "Kilo Code: Select Model"
3. Chọn model mong muốn từ danh sách:
   - Qwen3 Coder (Free)
   - GLM 4.5 Air (Free)
   - DeepSeek R1 (Free)
   - Kimi K2 (Free)

## Model hiện có

| Tên hiển thị | Model ID |
|-------------|----------|
| Qwen3 Coder (Free) | qwen/qwen3-coder-32b |
| GLM 4.5 Air (Free) | zhipu/glm-4.5-air |
| DeepSeek R1 (Free) | deepseek/deepseek-r1-0528 |
| Kimi K2 (Free) | moonshotai/kimi-k2 |

## Yêu cầu

- VS Code 1.70.0 trở lên
- Kilo Code extension đã được cài đặt và cấu hình model trong settings.json
